# Valentine Letter

## Overview
A cute, interactive Valentine's Day letter website. Users click an envelope to reveal a Valentine's question with animated cats and playful "Yes"/"No" buttons.

## Project Architecture
- **Type**: Static HTML/CSS/JS website
- **Files**:
  - `index.html` - Main page structure
  - `style.css` - Styling with heart background
  - `script.js` - Interactive logic (envelope open, button behavior)
  - Image assets: `envelope.png`, `cat_heart.gif`, `cat_dance.gif`, `yes.png`, `no.png`, `heart-bg.jpg`, `window.png`

## Running
- Served via Python's built-in HTTP server on port 5000
- Deployed as a static site from the root directory
